Kinematic Arm Plotter

For my robotic kinematics class, we had a closed-ended final project: we had to program a robotic arm to pick up and drop markers from the correct places and use them to draw 3 curves and 3 straight lines. However, to the dismay of my partners, I decided that we might as well make it so that it could draw anything.

To do that, I made the code take an SVG as input and process the points into a path that the arm could follow.
